查看命令所在源文件的位置
Created2023-03-06|Opslinuxgeneral
直接使用下面这条命令跟随符号链接来以获取实际的可执行文件 readlink -f `which java` readlink 命令会跟随一个符号链接。我在 which java 的外侧使用 readlink 将会使用 which java 的输出来替换要检查的符号链接，这被称之为命令替换。因此，在这个实例中，上面的命令大体上相当于 readlink -f /usr/bin/java 。 在我的示例中，可执行文件的位置是 /usr/lib/jvm/java-11-openjdk-amd64/bin/java 。
