Kafkaз”ҹдә§иҖ…гҖҒж¶Ҳиҙ№иҖ…зҡ„ж¶ҲжҒҜеҸҜйқ жҖ§ж–№жЎҲе®һзҺ°

Created2023-03-06|Updated2023-06-25|DevkafkaдёҺspringbootзҡ„ж•ҙеҗҲ
|Post View:|Comments:

д»ҘдёӢд»Јз ҒеҹәдәҺ SpringKafka 2.3.13.RELEASE + Boot 2.2.9.RELEASE е®һзҺ°

Producer ж¶ҲжҒҜзҡ„еҸҜйқ жҖ§

е®һзҺ°ж–№жЎҲпјҡack жЁЎејҸи°ғж•ҙ + йҮҚиҜ•жңәеҲ¶ + и§„йҒҝйҮҚиҜ•жңәеҲ¶дёӢеёҰжқҘзҡ„й—®йўҳ

з”ҹдә§иҖ…пјҡ

еҗҜз”ЁйҮҚиҜ•жңәеҲ¶еҗҺеёҰжқҘзҡ„й—®йўҳ

  1. йҮҚиҜ•иҝҮзЁӢдёӯпјҢдёҖжқЎж¶ҲжҒҜеҸӘдјҡеҗ‘еҗҢдёҖдёӘеҲҶеҢәиҝӣиЎҢйҮҚиҜ•еҸ‘йҖҒпјҢжүҖд»ҘеңЁйҮҚиҜ•зҡ„жңәеҲ¶дёӢпјҢд№ҹиғҪдҝқиҜҒж¶ҲжҒҜзҡ„е…ЁеұҖе№ӮзӯүжҖ§
  2. з”ұдәҺйҮҚиҜ•пјҢеҸҜиғҪеҜјиҮҙж¶ҲжҒҜеңЁ Node дёӯзҡ„йЎәеәҸе’Ң Producer еҸ‘йҖҒж—¶зҡ„йЎәеәҸдёҚдёҖиҮҙгҖӮеҸҜд»ҘеҜ№ max.in.flight.requests.per.connectioпјҲйҷҗеҲ¶жҜҸдёӘиҝһжҺҘпјҲжҢҮе®ўжҲ·з«ҜдёҺ Node д№Ӣй—ҙзҡ„ per.connection иҝһжҺҘпјүжңҖеӨҡзј“еӯҳ В е·ІеҸ‘йҖҒдҪҶжңӘж”¶еҲ°е“Қеә”зҡ„иҜ·жұӮж•°пјҢй»ҳи®Өдёә 5пјүи®ҫзҪ®дёә 1пјҢеҚіеҸҜдҝқиҜҒеңЁйҮҚиҜ•жңәеҲ¶дёӢзҡ„ж¶ҲжҒҜйЎәеәҸгҖӮ

Consumer ж¶ҲжҒҜзҡ„еҸҜйқ жҖ§

е®һзҺ°ж–№жЎҲпјҡжүӢеҠЁжҸҗдәӨ offset + йҮҚиҜ•жңәеҲ¶ + жӯ»дҝЎйҳҹеҲ—пјҲе‘ҠиӯҰпјү + жӯ»дҝЎйҳҹеҲ—ж¶ҲжҒҜйҮҮз”Ёе…¶д»–зӯ–з•ҘеҺ»еӨ„зҗҶж¶ҲжҒҜ
